# GeoMotion 功能规格

## 项目概述

GeoMotion 是一个面向教育场景的AI工具，帮助教师和学生更直观地理解和讲解几何题目。

**核心功能：**
1. 几何题 → 动图：拍照上传几何题目，自动生成解题动画
2. 草图 → 精确图：输入题目文本+手绘草图，生成印刷级精确图形

**目标用户：** 中小学数学教师、教辅资料编撰者

---

## 页面结构

```
/                    首页 - 上传入口
/analyze/[id]        分析结果页 - 显示解析结果和 GeoGebra 图形
```

## 功能1: 几何题 → 动图

### 输入
- 几何题目的图片（拍照/上传）

### 处理流程

1. **VLM 分析图片**
   - 识别图形类型（三角形、圆形、四边形等）
   - 生成 GeoGebra 命令
   - 提取已知条件
   - 识别求解目标
   - 给出解题步骤

2. **GeoGebra 渲染**
   - 嵌入 GeoGebra Applet
   - 执行命令生成图形
   - 用户可交互操作

3. **动画播放**
   - 逐行执行 GeoGebra 命令
   - 可控制播放速度、暂停、重置

### 输出
- GeoGebra 命令（可复制到 GeoGebra 使用）
- 可交互的几何图形（支持拖动、缩放、测量）
- SVG/PNG 导出

---

## 功能2: 草图 → 精确图

### 输入
- 题目文本描述
- 手绘草图照片（可选）

### 处理流程

1. VLM 理解草图意图和文本条件
2. 生成 GeoGebra 命令

### 输出
- GeoGebra 命令（可复制到 GeoGebra 使用）
- SVG/PNG 图形
